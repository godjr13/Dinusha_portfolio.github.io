Analog Synthesizer Design

Final Year Project

Final design for a sinewave synthesizer with a keyboard and octave control. Keyboard design was done using measurements from an actual keyboard. This can produce all the musical notes with a unique tone of it's own.

PCB design for voltage control sine wave generator


Designed a PCB for a Voltage-Controlled Oscillator (VCO) capable of generating stable sine wave outputs. The circuit is intended for analog synthesizer applications, with careful attention to component selection and layout for signal integrity and frequency stability. The design supports tuning via control voltage inputs.
